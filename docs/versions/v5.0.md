### Revolusi Visual

**Filosofi Desain:**
- Gaya editorial bercahaya (*Luminous editorial style*)
- Minimalis tapi premium
- Mode gelap sebagai default
- Tata letak terinspirasi majalah

**Sistem Warna:**
```css
Latar Belakang: #030303 (Hampir hitam)
Permukaan: #0f0f0f
Sorotan Permukaan: #1a1a1a
Primer: #ffffff
Aksen Mawar: #e11d48
Aksen Ungu: #a855f7
Aksen Zamrud: #10b981
```

**Suasana Latar Belakang:**
- Overlay butiran halus untuk tekstur
- Cahaya gradien yang menyebar
- Mode campuran layar (*Screen blend mode*) untuk kedalaman
- Efek denyut animasi (durasi 8 detik)
- Beberapa titik gradien untuk minat visual

**Tipografi:**
- Primer: Inter (300, 400, 500, 600)
- Aksen: Playfair Display (400, 600, 700, miring)
- Hierarki yang disempurnakan
- Keterbacaan yang ditingkatkan

- Largest Contentful Paint: 1.8s → 1.4s

**Optimasi:**

- Strategi optimasi gambar
- Ekstraksi CSS kritis
- Pemuatan awal font
- Petunjuk sumber daya (*preconnect*)

### Sistem Animasi

**Penggunaan Framer Motion:**
- Transisi halaman dengan AnimatePresence
- Animasi masuk komponen
- Interaksi hover
- Animasi berbasis gulir

**Mode Animasi:**
```typescript
<AnimatePresence mode="wait">
  <Routes location={location} key={location.pathname}>
    {/* Rute dengan transisi halaman */}
  </Routes>
</AnimatePresence>
```

### Perilaku Gulir


**Gulir ke Atas:**

## Panduan Migrasi

### Dari v4.x ke v5.0

**Perubahan yang Merusak (*Breaking Changes*):**
- Sistem warna direvisi total
- API komponen diperbarui (minor)
- Sistem latar belakang diubah

**Langkah-langkah:**
1. Perbarui referensi warna
2. Periksa props komponen
3. Uji animasi
4. Verifikasi perilaku responsif
5. Validasi aksesibilitas
