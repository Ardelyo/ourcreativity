# OurCreativity

OurCreativity adalah proyek open source, tetapi penggunaan nama, logo, dan identitas merek OurCreativity memiliki batasan tertentu demi menjaga brand kami.

## Lisensi Penggunaan Kode

Source code pada repository ini menggunakan lisensi open source (lihat pada file `LICENSE`). Anda bebas untuk membaca, memodifikasi, dan mendistribusikan ulang kode, **namun dengan ketentuan berikut:**

> **Penggunaan nama, logo, grafik, atau identitas "ourcreativity" pada produk turunan, fork, atau distribusi ulang TIDAK DIPERBOLEHKAN tanpa izin tertulis dari pemilik repositori.**

Contoh yang DILARANG:
- Menggunakan nama "ourcreativity" pada project/fork/modifikasi Anda.
- Menggunakan logo ourcreativity/branding dalam rilis Anda.

Contoh yang DIPERBOLEHKAN:
- Mengubah nama dan logo untuk produk turunan Anda.
- Menyebutkan "fork from ourcreativity" hanya pada catatan sejarah/README, tanpa menyalahgunakan branding.

## Aturan Penerapan

Penggunaan kode: **Bebas, sesuai lisensi open source yang tertera.**  
Penggunaan branding "ourcreativity": **MUSTI ada izin**.

Jika ingin menggunakan nama atau identitas merek OurCreativity untuk keperluan publikasi, komersial, atau distribusi, silakan kontak kami via [issues](https://github.com/Ardelyo/ourcreativity/issues) atau email yang tertera.
